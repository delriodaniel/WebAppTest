PROGRAMA PREGUNTAS

  • + - Solicitado por el cliente

    • El cliente solicita la creación de una aplicación para facilitar la comprensión de conceptos.

      La aplicación debe parecerse a una prueba de tipo test. Por lo general se pondran cuatro respuestas por pregunta, siendo valida solo una de ellas, a excepción de aquellas que sean de verdadero o falso (solo se mostraran dos opciones).

      Cada respuesta correcta o incorrecta se grabara para posteriormente hacer una estadística. Dicha estadística se utilizara para reflejar que preguntas son las que se fallan más y así crear un algoritmo para incidir en ellas, de este modo se facilita la adquisición de los conocimientos.

  • + - Proceso de instalación
    • + - Una vez tengamos la aplicación completa y funcional se han de seguir los siguientes pasos.

      • Se debe contratar un servicio de hosting que asegure el buen servicio de los servidores, de este modo se podra acceder desde cualquier lugar con conexión a la red.

        Nos aseguramos de que el archivo que controla la página inicial de nuestra aplicación tenga un nombre que sea favorable para su carga automática en la red, como podría ser index.php, dependiendo de la configuración que tengamos.

        + - Accederemos a la página de nuestro proveedor para comenzar la importación y o creación de las bases de datos pertinentes y carpetas necesarias para nuestra aplicación

        • + - En el momento de subir los archivos podemos tener varias opciones para completar la operación, esto depende de nuestro proveedor. En este caso tenemos varias opciones

          • Dentro de nuestro perfil de cliente, con la/s carpeta/s creada/s, se nos permitira subir los archivos de la aplicación de uno en uno.

            Como segunda opción tenemos la posibilidad de usar un servicio FTP para conectar con el hosting, con este sistema podemos subir una mayor cantidad de archivos al mismo tiempo. FTP contempla la posibilidad de guardar la conexión al hosting, facilitando el acceso para subir futuras modificaciones de los archivos.

        Tras subir los archivos mediante el sistema que elijamos pasaremos a comprobar que la aplicación esta subida correctamente, esto quiere decir que todo este bien relacionado con la base de datos y su funcionamiento sea igual de bueno que antes de subirlo.

        Llegados a este punto solo quedara revisar de vez en cuando, a medida que crezcan las estadísticas, la cantidad de usuarios y se realicen actualizaciones.

  • + - Pantalla inicial, menu seleccion
    • Menu dinamico en el cual se muestran las aplicaciones actualmente disponibles para su uso
  • + - Inicio de la aplicacion de repaso, registro

    • Pantalla obligatoria, solo apareceran las opciones, de registrarse y para poder acceder .

      Tras registrarse o acceder como usuario ya registrado aparece una pantalla de bienvenida para llevarte al menu de la aplicacion

  • + - menu navegacion de la aplicacion de repaso

    • Presentara elementos comunes en toda la aplicacion: el usuario con el que se accede, la opcion de cerrar sesion, junto a un pequeño menu para seleccionar entre cuatro opciones(introducir preguntas, listar preguntas, iniciar las preguntas y ver las estadisticas generales).

      Introduccion de nuevas preguntas

      • Aparecera un formulario, en el cual esta detallado que datos son necesarios para poder introducir una nueva pregunta

        menu de navegacion simple, con las opciones de enviar pregunta(una vez que los campos necesarios esten completos) y de volver a la pantalla de navegacion de la aplicacion de preguntas

        • En caso de que se cree el registro aparecera una pantalla de control mostrando el exito de la operacion, en caso contrario mostrara el fracaso de la misma

      Listado de preguntas

      • Se mostrara una tabla en la cual se muestran las preguntas y sus multiples respuestas teniendo marcada aquella que es correcta(con un fondo verde)

        Volvera a contener un par de botones dando la opcion de volver a la pantalla anterior o de crear una nueva pregunta

      inicio de la ronda de preguntas

      • Llegara a una pagina que muestra pregunta a pregunta todos los registros que contiene
      • Creada a modo tipo test para facilitar la interaccion

        • Al responder se generara una pantalla de control que mostrara los datos que se pasan y mostrara si la respuesta es la correcta o no.

      • Las preguntas se mostraran aleatoriamente siguiendo un algoritmo dependiente de las estadisticas

      Estadisticas

      • Las estadisticas generales se mostraran en formato tabla señalando en cada fila la pregunta y las veces que han acertado o fallado los usuarios en dicha pregunta
